# Nebula Vortex

Mobile-first single-file vertical space bullet hell game.

GitHub Pages: https://brandon-valley.github.io/nebula-vortex-bullet-hell/

## How to play

- Drag anywhere on the screen to move.
- Shooting is automatic.
- Graze enemy bullets for score multiplier and bomb charge.
- Tap BOMB or double tap the screen when the gold meter is full.

This repo publishes `index.html` through GitHub Pages on every push.
